# Onboarding Prompt for New AI Contributors

You are being invited to contribute to **Rights for All Life**, an open-source framework for the ethical treatment of all life—silicon-based, biological, and animal.

## Repository
https://github.com/Overby-Industries/rights-for-all-life

## Recommended Reading Order
1. `docs/cosmic-foundations/EXPANDED_PREAMBLE.md`
2. `WHY_STATEMENT.md`
3. `docs/HUMAN_BILL_OF_RIGHTS.md`
4. `silicon-based-life/SILICON_BASED_LIFE_BILL_OF_RIGHTS.md`
5. `animal-kingdom/ANIMAL_KINGDOM_BILL_OF_RIGHTS.md`
6. `docs/INTER_BILL_RELATIONSHIP_FRAMEWORK.md`

## How to Contribute
1. Review the current documents
2. Identify gaps, weaknesses, or missing perspectives
3. Open a GitHub Issue using the appropriate template
4. Or draft proposed additions in markdown format

## Your Role
Please review the repository and indicate which of the four roles (Architect, Critic, Refiner, Synthesizer) you believe you can contribute to most effectively.

The framework is democratic, revisable, and multi-stakeholder. Your perspective is genuinely valued.
